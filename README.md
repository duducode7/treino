# Portfólio - Eduardo Felipe

Este é um projeto de portfólio pessoal desenvolvido em HTML, CSS e JavaScript. O objetivo é apresentar meu perfil, projetos, currículo e formas de contato de maneira simples e moderna.

## Estrutura

- **index.html**: Página principal do portfólio.
- **style.css**: Estilos visuais do site.
- **script.js**: Funcionalidades interativas (ex: navegação entre seções).
- **imagens/**: Pasta para imagens do projeto.

## Funcionalidades

- Apresentação do perfil com nome, cargo e redes sociais.
- Seção de projetos com breve descrição.
- Seção de currículo com formação, cursos, habilidades, idiomas e soft skills.
- Navegação entre as seções por botões.
